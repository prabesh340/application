## What does this website have?
- **Modern Landing Page**: Eye-catching hero section, smooth scroll, and animated effects
- **Shop/Products**: Browse all Lexi flavors, see product details, and add to cart
- **Product Pages**: Each drink has a dedicated page with images, description, price, and quantity selector
- **Gallery**: Explore Lexi in action with a dynamic, scrollable image gallery
- **About Us**: Learn about the Lexi story, mission, and vision
- **Contact**: Reach out to the Lexi team directly from the site
- **Responsive Design**: Fully optimized for mobile, and desktop
- **Fast & Smooth**: Built with Next.js 16, React 19, and Tailwind CSS 4 for top performance
- **GSAP Animations**: Engaging transitions and effects throughout the site
- **Sustainable & Community-Driven**: Lexi is committed to quality, sustainability, and empowering its community

- **Framework**: Next.js 16 (App Router)

- **Other**: React 19, Swiper, Lenis, Hamburger-react, Lucide-react

## Getting Started
1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
3. **Open** [http://localhost:3000](http://localhost:3000) **to view the site.**
